

web3 作用：和 eth节点 进行交互


web3 有多种语言版本：

- js [web3.js](https://github.com/ethereum/web3.js)
- [Python Web3.py](https://github.com/ethereum/web3.py)
- Haskell hs-web3
- Java web3j
- Scala web3j-scala
- Purescript purescript-web3

开发者可以选择使用自己善长的语言进行开发。
